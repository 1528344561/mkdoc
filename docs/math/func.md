## 积分

### $\Gamma$ 函数

$\Gamma(\alpha) = \int_{0}^{+\infty} x^{\alpha-1}e^{-x}dx =^{x=t^2}2\int_{0}^{+\infty} t^{2\alpha-1}e^{-t^2}dt$

$\Gamma(1) = \Gamma(2) = 1$ $\ \ \ \ \ \Gamma(\frac{1}{2}) = \sqrt{\pi}$

$\Gamma(1+\alpha) = \alpha\Gamma(\alpha)$
